CSC 380 Project 2 - Secure(?) Chat

Concept

This project aims to develop both asymmetric and symmetric encryption systems for secure communication, with a focus on implementing perfect forward secrecy and preventing replay attacks. The system utilizes RSA for asymmetric encryption, leveraging the GMP library for handling arbitrary precision numbers. Simultaneously, a symmetric encryption system labeled as SKE is constructed, functioning exclusively on buffers and files. Key features include:

  • Implementation of a handshake protocol to establish ephemeral keys with perfect forward secrecy.
  • Mutual authentication using public key cryptography.
  • Encryption of each message with authentication tagging to ensure integrity.
  • Measures to prevent replay attacks.

Compiling the Skeleton

To compile the provided skeleton, follow these steps:

  1. Navigate to the directory containing the skeleton code.
  2. Run the compilation command, e.g., make.
  3. Once compiled, run ./chat -h to view a list of available options.
  4. You can test the chat functionality by running the command:
$ ./chat -l & sleep 1 && ./chat -c localhost &

This will open two windows for chat clients. Note that they may initially appear on top of each other, so you may need to move one window to see the other.
